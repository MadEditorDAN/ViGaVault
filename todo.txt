Bug/Features :

-Create a new windows called by the menu tools/Game manager.In that windows we will find a Batch game edit, Batch game delete. 
In that windows, List the games in a table with the following columns: release date/name/platfoms/filter category 1/filter category 2, etc.../developper/publisher. On top of each column put listbox with the content of the below column, when you select an entry it filter on that entry, and you can select multiple entries  (like the excel filter box).
On top of the name column, a text box. If you type in it, it search the game containing what you typed, and the list dynamically shrink to that selection. 
If you select something in the upper listbox it filter the below list to what you selected.
Also, and a custom word list user can create to filter the gamelist on (goodie, dlc,...) so non games get removed from the list. Also change the main counter total in the sidebar and remove those excluded games from the count.

- allow the possibility to change the horizontal size of the sidebar from 1/5 (moe or less actual size) to max 1/3 of the full windows. 
To adapt the content, don't change anything but the number of columns in the filters. It will adapt from 2 to 3 or 4 (depending on the available space)
