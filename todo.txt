Voici les informations techniques que tu dois donner à Gemini pour qu'il code le connecteur :
Source de données : API Store Front de Steam.
Endpoint : https://store.steampowered.com/api/appdetails?appids={APPID}&l=french

Données à extraire (Metadata) :
---------------------------------------------------------------------------------------------------------
Type,		Champ Steam API,									Action ViGaVault
---------------------------------------------------------------------------------------------------------
Metadata,	"name, developers, publishers, release_date",		Injection CSV via LibraryManager
Texte,		"about_the_game, short_description",				Récupéré dans la langue de l'UI
Genres,		genres,												Passage par ta fonction normalize_genre
Images,		"header_image (Cover), 								background",Téléchargement dans ./images/
Vidéos,		movies -> mp4 -> max,								Téléchargement .mp4 si lien direct
Vidéos,		YouTube URL,										Copie simple du lien dans le CSV
---------------------------------------------------------------------------------------------------------
Si un lien .mp4 ou .webm existe (ex: mp4['max']) -> Téléchargement direct dans ./videos/.
Si seul un lien YouTube est présent -> Copie de l'URL dans le CSV uniquement.

3. Protocole de Connexion (Le Flux)

Récupération de la liste : On utilise l'URL userdata de Steam (nécessite les cookies de session de l'utilisateur) pour obtenir tous les AppIDs possédés. 
Boucle d'enrichissement : Pour chaque AppID, on appelle l'API appdetails pour obtenir le "gros" JSON.
Injection : On passe ce dictionnaire à LibraryManager.add_game_to_db pour qu'il s'occupe de l'écriture dans VGVDB.csv et du rangement des fichiers.

5. Pourquoi commencer par Steam ?
Fiabilité : L'API appdetails de Steam est la plus documentée et la plus stable du marché.
Qualité : C'est là que tu trouveras les plus belles descriptions et les trailers les plus faciles à télécharger (souvent en MP4 direct sans protection).
Template : Une fois que Steam fonctionne, on pourra copier 80% du code pour GOG et Epic, car la logique de "Récupérer ID -> Chercher Infos -> Télécharger Médias" sera déjà en place.
