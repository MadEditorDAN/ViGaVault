ViGaVault User Guide

ViGaVault allows you to manage your local game collection alongside GOG Galaxy integration.

Basic Usage:

  • Scanning: Use 'Tools > Full Scan' or the 'FULL Scan' button to detect games.
  • Editing: Click the 'Edit' (pencil) button on a game card to modify details.
  • Media: ViGaVault automatically fetches covers. Trailers can be downloaded in Settings.
  • Filtering: Use the sidebar to filter by Platform, Genre, etc.

This is a beta version. More features coming soon!
